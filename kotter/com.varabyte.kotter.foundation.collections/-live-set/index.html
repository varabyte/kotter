LiveSet

@ThreadSafe
class LiveSet<T> : MutableSet<T>

Like LiveVar, but for sets.

In other words, adding or removing to the set will cause the active section to rerender automatically.

For example:

val ids = liveSetOf<String>()

section {
  for (id in ids) { ... }
}.run {
  ids.add(...)
}
