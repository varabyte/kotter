fun <T : Any> remove(key: ConcurrentScopedData.Key<T>): Boolean

A simple remove option which returns true if the value was removed or not. If removed, its dispose call would have been triggered.

fun <T : Any> remove(key: ConcurrentScopedData.Key<T>, block: T.() -> Unit)

Attempt to remove a key directly, which will trigger its dispose call if present. If a value was removed, it will additionally be triggered in the passed-in block.
