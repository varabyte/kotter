OffscreenBuffer

class OffscreenBuffer

An internal buffer that stores commands without actively rendering them.

As a user, you won't create one yourself, but you'll use offscreen to do so.

This class itself is inert, but you can use createRenderer to create another class that can apply its commands to the current render scope.
